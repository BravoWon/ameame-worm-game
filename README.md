# 🐛 Ameame's Worm Adventure

A cute kawaii 3D worm game — steer your wiggly worm and eat candy to grow!

**▶ Play it:** https://bravowon.github.io/ameame-worm-game/

## How to play
- Steer with the **← →** arrow keys (or **A** / **D**), or tap the left/right side of the screen.
- Gobble the candy 🍬 to grow longer and rack up points.
- There's no losing — just wiggle around and grow as long as you like 💕

Built with [Three.js](https://threejs.org) in a single self-contained `index.html`.
